## Current Direction

ThoughtWeave is currently being built as a ChatGPT App using OpenAI Apps SDK.

ChatGPT remains the conversation surface. ThoughtWeave renders an adaptive companion widget from user/model-provided context through MCP tools.

The legacy Electron prototype remains in the repo as a reference implementation for the earlier Research Review layout.
